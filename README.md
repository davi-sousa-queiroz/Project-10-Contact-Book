# (Project 10) Contact Manager 📇 (March 7th-8th 2026)

This project is a simple contact manager built in Python.  
It allows users to store and manage contacts using dictionaries.

The program runs in the terminal and provides a menu where users can add, search, view, and delete contacts.

## Features

- Add a new contact
- View all contacts
- Search for a contact
- Delete a contact
- Menu-based interface

## How It Works

Contacts are stored in a Python dictionary.

Example structure:

name → phone number

The program uses loops and functions to manage user input and perform actions on the contact list.

## Example

Add a contact:

Name: John  
Phone: 555-1234

The program stores it and allows you to search or display it later.
